BioStudio

Created by Manish Kumar Gupta

The BioStudio is a powerful web platform designed to simplify and streamline your bioinformatics and data analysis workflows. It currently integrates multiple specialized applications — each focused on distinct areas such as machine learning predictions, pipeline management, and genome visualization — all accessible from a unified home interface.

Select an application below to get started:

  • Pipeline Manager — Create, monitor, and manage complex data processing pipelines with ease.
  • IGV Viewer — Visualize genomic data interactively using an integrated genome browser.
  • ML Predictor Upload — Upload your biological datasets and get quick machine learning-based predictions.
  • Variant Annotation — Annotate genetic variants with relevant biological information.
  • Gene Annotation — Annotate genes with relevant biological information.
  • Pathway Enrichment — Analyze biological pathways enriched in your dataset.
  • Literature Summarizer — Summarize scientific literature relevant to your research.
  • Network Analysis — Explore and analyze biological interaction networks.
  • Single Cell Analysis — Visualize and analyze single cell RNA-seq data, including UMAP and gene expression plots.
